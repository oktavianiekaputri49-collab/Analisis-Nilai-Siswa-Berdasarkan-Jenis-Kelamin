📊 Analisis Nilai Siswa Berdasarkan Jenis Kelamin

Deskripsi
Proyek ini berfokus pada analisis data nilai siswa menggunakan Microsoft Excel. Data dianalisis berdasarkan jenis kelamin untuk mengetahui perbandingan rata-rata nilai antara siswa laki-laki dan perempuan.

Dataset
Data terdiri dari:
- Nama siswa
- Jenis kelamin (Laki-laki / Perempuan)
- Nilai

Tools & Fungsi yang Digunakan
- Microsoft Excel
- Fungsi AVERAGE untuk menghitung rata-rata
- Pengelompokan data berdasarkan kategori

Hasil Analisis
- Rata-rata nilai siswa laki-laki: 57.6
- Rata-rata nilai siswa perempuan: 74.4
- Terdapat perbedaan nilai rata-rata antar kelompok

Tujuan Proyek
- Melatih pengolahan data statistik sederhana
- Memahami penggunaan fungsi AVERAGE di Excel
- Menganalisis data berdasarkan kategori

Insight
Perbedaan rata-rata nilai menunjukkan variasi hasil belajar antar kelompok. Analisis sederhana ini membantu memahami distribusi data dengan lebih mudah.

Penutup
Proyek ini merupakan latihan dasar analisis data statistik menggunakan Excel.
